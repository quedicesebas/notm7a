# **App Name**: Defendamos la Séptima - Comité Ciudadano

## Core Features:

- Committee Information Display: Showcase core information about the Defendamos la Séptima - Comité Ciudadano, including its mission and objectives. An AI tool can help generate concise summaries from provided social media content or general themes for the 'About' section, to be displayed on the static Angular application.
- Social Media Links: Clearly display links to the committee's social media profiles (X and Instagram) for easy access to latest updates and community engagement, implemented within the static Angular app.
- Donation Call-to-Action: A prominent button allowing users to initiate a donation. The button, built with Angular, will open a specific URL in a modal or embedded frame within the static page.
- Modal Donation Form: Integrate the donation URL into a clean modal window or an inline frame, implemented using Angular, to facilitate contributions directly on the page without requiring a separate backend.
- Responsive Design: Ensure the Angular-based landing page is fully responsive and accessible on various devices, from mobile phones to desktop computers, for a consistent user experience.

## Style Guidelines:

- Primary color: A bold red (#D82020) derived from the logo, reflecting the committee's central message of action and defense.
- Background color: A clean, light gray (#F8F8F8) to provide a neutral backdrop that makes the primary red and accent yellow pop, enhancing readability.
- Accent color: A vibrant yellow (#FFD700) directly from the logo, used for key interactive elements like the 'Donate' button, symbolizing attention and hope.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable qualities, ensuring that information is conveyed with clarity and professionalism.
- Utilize simple, outline-style icons for navigation and social media links, maintaining a modern and clean aesthetic that complements the overall design.
- A clean, single-page layout with clear sections for committee information, social media links, and the donation call-to-action, prioritizing ease of navigation and conversion.
- Subtle hover effects on buttons and links, and a gentle fade-in animation for the donation modal, providing user feedback and a polished feel without being obtrusive.
