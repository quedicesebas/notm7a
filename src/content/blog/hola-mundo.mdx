# Bienvenidos a nuestro nuevo Blog

Este es el primer artículo del blog de **Defendamos la Séptima**. El propósito de este espacio es compartir ideas, actualizaciones y propuestas urbanas que beneficien a toda la ciudad, sin colores políticos.

## ¿Qué es el MDX?

El MDX es una forma de escribir contenido (como este texto) de forma muy sencilla, pero con la **súper ventaja** de poder incluir elementos interactivos directamente desde el código de nuestra página web.

Por ejemplo, aquí hay una lista de las cosas que defendemos:
- Independencia técnica y política.
- Opciones reales como trenes ligeros y buses eléctricos.
- Defensa del patrimonio urbano y los árboles.
