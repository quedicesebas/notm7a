# Comité Ciudadano Defendamos la Séptima

Sitio web oficial del **Comité Ciudadano Defendamos la Séptima** ([defendamoslaseptima.org](https://defendamoslaseptima.org)). Un corredor vial digno, sostenible y centrado en las personas para Bogotá.

## 🚀 Características
- **Landing Page Informativa:** Secciones de inicio, nosotros, objetivos y redes sociales.
- **Participación Ciudadana:** Integración con plataformas de donación (Arma tu Vaca).
- **Firebase Integration:** Google Analytics para rastreo de eventos y despliegue rápido.
- **Diseño Premium:** Interfaz moderna usando Tailwind CSS y Radix UI.

## 🛠️ Stack Tecnológico
- **Framework:** [Next.js 15](https://nextjs.org/)
- **Estilos:** Tailwind CSS
- **Componentes:** Radix UI / Lucide React
- **Backend/Analytics:** Firebase

## 💻 Desarrollo Local

Para ejecutar el proyecto localmente:

1. Instala las dependencias:
   ```bash
   npm install
   ```

2. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Abre [http://localhost:9002](http://localhost:9002) en tu navegador.

## 🚢 Despliegue
El proyecto está configurado para desplegarse en Firebase:
```bash
npm run deploy
```
